Carreras en Datos
Ingeniero/a de Datos
Función
Construye y gestiona infraestructuras que permiten mover, transformar y almacenar datos.
Habilidades
SQL, Python, bases de datos, lógica, arquitectura de datos.
Herramientas
Apache Airflow, Spark, dbt, AWS, GCP, Azure.
Analista de Datos
Función
Analiza datos para generar reportes que apoyen decisiones de negocio.
Habilidades
Excel, SQL, visualización, pensamiento crítico.
Herramientas
Excel, Power BI, Tableau, SQL, Python (básico).
Científico/a de Datos
Función
Crea modelos predictivos y extrae insights complejos a partir de los datos.
Habilidades
Estadística, programación, machine learning.
Herramientas
Python, R, Jupyter, scikit-learn, SQL, Git.
Analista de Negocios
Función
Conecta los datos con decisiones estratégicas dentro de la organización.
Habilidades
Comunicación, lógica de negocio, análisis de datos.
Herramientas
Excel, Power BI, SQL, herramientas de gestión (Jira, Trello).
Ingeniero/a de Machine Learning
Función
Desarrolla, entrena y despliega modelos de machine learning para producción.
Habilidades
Programación, ML avanzado, despliegue de modelos.
Herramientas
Python, TensorFlow, PyTorch, Docker, MLflow.
Ingeniero/a en IA Generativa
Función
Crea aplicaciones con modelos generativos como GPT o Stable Diffusion.
Habilidades
Procesamiento de lenguaje natural (NLP), prompt engineering, uso de APIs.
Herramientas
Python, OpenAI, LangChain, Hugging Face, Git.
Ingeniero/a de Big Data
Función
Diseña y mantiene sistemas para procesar y analizar grandes volúmenes de datos en tiempo real o por lotes.
Habilidades
Procesamiento distribuido, arquitectura escalable, bases de datos NoSQL.
Herramientas
Hadoop, Spark, Kafka, Hive, Flink, Scala, Python, servicios en la nube.
